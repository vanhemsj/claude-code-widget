# Claude Code Widget

A lightweight floating widget for Claude Code.

![Screenshot](screenshot.png)

## Requirements

- Linux - Windows 10/11
- Active Claude Code subscription

## Features

- **Usage** - Monitor your Claude Code API usage in real-time

## Installation

Download the latest installer from [Releases](https://github.com/vanhemsj/claude-code-widget/releases).

## Development

```bash
# Install dependencies
npm install

# Run the app
npm start

# Build installer
npm run build
```
